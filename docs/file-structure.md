# File Structure - Project Layout

## Purpose
Complete directory structure of the Trotid Shell Quickshell project and associated dotfiles.

## Quickshell Source (`~/Desktop/Trotid_Shell/quickshell/`)
```
quickshell/
├── shell.qml                       # Entry point - all PanelWindows + GlobalShortcuts
├── BarContent.qml                  # Status bar layout (left/center/right)
├── Caching.qml                     # Cache/state/run directory management
├── calendar/
│   ├── .env                        # OpenWeatherMap API config
│   └── weather.sh                  # Weather data fetcher + cache
├── core/
│   └── NotificationUtils.js        # Time formatting, icon resolution, icon mapping
├── functions/
│   └── ColorUtils.qml              # HSV/HSL color manipulation utilities
├── services/
│   ├── qmldir                      # Singleton registrations
│   ├── ShellState.qml              # UI state management (activePopup pattern)
│   ├── AudioService.qml            # wpctl status parsing, sink/source/BT
│   ├── VolumeService.qml           # Volume control + mute toggle
│   ├── BrightnessService.qml       # Brightness polling + control
│   ├── NetworkService.qml          # WiFi status + speed monitoring
│   ├── BatteryService.qml          # UPower battery + health
│   ├── SystemService.qml           # CPU/memory/temp from /proc
│   ├── NotificationService.qml     # DBus notification server + persistence
│   └── ColorService.qml            # Material You colors from colors.json
└── widgets/
    ├── BluetoothSelector.qml       # Orbiting-card BT device manager
    ├── WifiSelector.qml            # Orbiting-card WiFi network manager
    ├── NotificationPanel.qml       # Grouped notification history
    ├── NotificationPopup.qml       # Toast notification display
    ├── Cheatsheet.qml              # Searchable keybind reference
    ├── WallpaperPicker.qml         # Coverflow wallpaper selector
    ├── QuickActions.qml            # Screenshot/recording quick HUD
    ├── OsdPopup.qml                # Volume/brightness OSD
    ├── CalendarPopup.qml           # Clock + calendar + weather
    ├── Colors.qml                  # (Color definitions, legacy)
    ├── MediaCard.qml               # MPRIS media player card
    ├── PlayerCard.qml              # (Alternate player card)
    ├── WaveVisualizer.qml          # (Audio visualizer)
    └── WorkspaceOverview.qml       # (Workspace overview)
```

## Dotfiles (`dotfiles/`)
```
dotfiles/
└── .config/
    ├── hypr/
    │   ├── hyprland.conf           # Main Hyprland config (sources keybinds.conf)
    │   └── keybinds.conf           # All keybinds in one file
    ├── scripts/
    │   ├── screenshots/
    │   │   └── screenshot.sh       # Screenshot helper (grim+slurp)
    │   ├── recording/
    │   │   └── recording.sh        # Screen recording (wf-recorder)
    │   └── clipboard-picker.sh     # Clipboard history (cliphist+rofi)
    ├── kitty/
    │   └── kitty.conf              # Terminal config (references wallust cache)
    ├── rofi/
    │   ├── launchers/              # App launcher styles
    │   └── applets/                # Applet styles (used by recording.sh)
    ├── wallpapers/                 # Wallpaper images directory
    └── waybar/                     # Legacy waybar config (not active)
```

## Installed Locations
```
~/.config/quickshell/
├── mrtrotid-shell/    → ~/Desktop/Trotid_Shell/quickshell/ (symlink)
└── custom/            → ~/Desktop/Trotid_Shell/quickshell/ (symlink)

~/.config/scripts/     → ~/Desktop/Trotid_Shell/scripts/ (symlink)
~/.config/wallpapers/  → Wallpaper images directory
```

## Cache Locations
```
~/.cache/quickshell/
├── colors.json                        # Generated by matugen
├── notifications.json                 # Persisted notifications
├── weather/
│   └── weather.json                   # Cached weather data (15min TTL)
└── wallpaper_picker/
    ├── thumbs/                        # Pre-generated wallpaper thumbnails
    ├── colors_markers/                # Extracted dominant colors
    └── current_wallpaper.png          # Current wallpaper thumbnail
```

## Config Locations
```
~/.config/matugen/
└── templates/                         # Matugen color templates (edit these)

~/.config/wallust/
└── templates/                         # Wallust terminal color templates

~/.config/hypr/
├── hyprland.conf                      # Main config
├── keybinds.conf                      # All keybinds
└── colors/                            # Generated by wallust (DO NOT EDIT)
```

## Key File Relationships
```
shell.qml
├── imports services/ (all singletons via qmldir)
├── imports widgets/ (all popup components)
└── creates PanelWindows for each popup

BarContent.qml
├── binds to ColorService.* (Material You)
├── binds to SystemService.* (CPU/mem/temp)
├── binds to VolumeService.* (volume)
├── binds to AudioService.* (sinks/mic)
├── binds to BrightnessService.* (brightness)
├── binds to NetworkService.* (WiFi/speed)
├── binds to BatteryService.* (battery)
├── binds to NotificationService.* (unread)
└── binds to ShellState.* (popup toggles)

ColorService.qml
├── reads ~/.config/quickshell/mrtrotid-shell/colors.json
└── generated by matugen from wallpaper
```

## Modifying the Structure
- Add new service: Create in `services/`, register in `services/qmldir`
- Add new widget: Create in `widgets/`, add PanelWindow in `shell.qml`
- Add new keybind: Add to `keybinds.conf`, optionally add GlobalShortcut in `shell.qml`
- Add new script: Create in `scripts/`, add keybind in `keybinds.conf`
