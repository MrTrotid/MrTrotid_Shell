# Trotid Shell — Dotfiles

Deployable configuration files for the Trotid Shell desktop environment.

## Structure

```
dotfiles/.config/
├── hypr/                         # Hyprland configuration (Lua API)
│   ├── hyprland.lua              # Main entry — env, input, general, decoration,
│   │                             #   animations, gestures, autostart
│   ├── hyprland.conf             # Fallback .conf (auto-removed if .lua exists)
│   ├── monitors.lua              # Auto-generated monitor specs + workspace assignments
│   ├── monitors.conf             # Fallback .conf monitor config
│   ├── windowrules.lua           # Window rules + layer rules (Lua API)
│   ├── colors/
│   │   ├── colors.lua            # Material You color variables (generated by wallust)
│   │   └── colors.conf           # Hyprlang color variables (generated by wallust)
│   └── configurations/
│       ├── keybinds.lua           # All keybindings (single file for cheatsheet)
│       └── keybinds.conf          # Hyprlang keybinds fallback
├── quickshell/
│   ├── mrtrotid-shell/           # Main shell (symlinked to ~/.config/quickshell/)
│   │   ├── shell.qml             # Root — all PanelWindows + GlobalShortcuts
│   │   ├── BarContent.qml        # Bar layout
│   │   ├── Caching.qml           # Cache/state directory management
│   │   ├── core/                 # JavaScript utilities
│   │   ├── services/             # Singleton services (qmldir + pragma Singleton)
│   │   ├── widgets/              # Popup components
│   │   ├── calendar/             # Weather scripts + .env
│   │   └── functions/            # Color utilities
│   └── overview/                 # Workspace overview config override
├── rofi/
│   ├── launchers/                # App launcher styles (type-1..style-6)
│   └── applets/                  # Applet styles
├── kitty/
│   └── kitty.conf                # Terminal config (references wallust cache)
├── wallpapers/                   # Wallpaper images (scanned by picker)
└── local/
    └── bin/                      # Executable scripts (copied to ~/.local/bin/)
        ├── wallset               # Wallpaper selector (rofi)
        ├── wallset-backend       # Theming pipeline (swaybg, wallust, matugen)
        ├── wallset-backend-startup # Random wallpaper on login
        └── ...                   # Additional utilities
```

## Key Components

### Hyprland (Lua API)

The Lua config uses `dofile()` to include modular configs:

```lua
dofile("colors/colors.lua")
dofile("monitors.lua")
dofile("windowrules.lua")
dofile("configurations/keybinds.lua")
```

If both `hyprland.lua` and `hyprland.conf` exist, the installer removes `.conf` to avoid conflicts.

### Quickshell (mrtrotid-shell)

Config name: `mrtrotid-shell` — launched via `quickshell -c mrtrotid-shell`.

Installed location: `~/.config/quickshell/mrtrotid-shell/` (symlink to repo for live hot-reload).

### Singleton Services

All in `quickshell/mrtrotid-shell/services/` with `pragma Singleton` + `qmldir`:

- **ShellState** — UI toggle states (activePopup mutual exclusion)
- **ColorService** — Reads matugen colors.json via Process + cat (2s polling)
- **AudioService** — wpctl parsing, sink switching, BT auto-switch, mic
- **BrightnessService** — brightnessctl polling + control
- **VolumeService** — Volume from AudioService, debounced refresh
- **NetworkService** — nmcli monitoring, /proc/net/dev speed
- **BatteryService** — UPower + sysfs, low battery warnings
- **SystemService** — CPU + memory + temperature from /proc
- **NotificationService** — DBus notification server, grouped, toasts, persistence
